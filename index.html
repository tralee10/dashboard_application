Seattle Vehicle Collisions Reset Map

This dashboard visualizes vehicle collisions in Seattle from 2016-2020 using proportional symbols. Different colors represent the collision severity. Click on a point to view detailed collision information.

Collisions in Displaying Region

The diagram shows the total number of collisions that occured. The bars show the number of collisions depending on the number of injuries from an accident.
